Directives

DuVay uses HTML-first data-w-* hooks for progressive behavior. They serve the same purpose as directives in heavier frameworks, but work in plain documents.

Theme

Use data-w-theme-toggle to cycle the root data-w-theme value.

<button class="w-btn w-btn-outlined" data-w-theme-toggle> Toggle theme </button>

Menus and navigation

HookBehavior
data-w-dropdown-triggerOpens or closes the matching data-w-dropdown target.
data-w-section-toggleCollapses or expands the nearest sidebar section.
data-w-sidebar-toggleToggles the mobile sidebar overlay.
data-w-searchMarks the global search input for keyboard focus behavior.

Dialogs and sheets

Open and close targets by id or by matching data-w-dialog / data-w-sheet. Add data-w-overlay on the target when it owns a named overlay.

<button class="w-btn w-btn-filled" data-w-dialog-open="confirm">Open</button> <div id="confirm" class="w-dialog-wrapper" data-w-dialog="confirm" aria-hidden="true"> <div class="w-dialog"> <button class="w-btn w-btn-text" data-w-dialog-close="confirm">Close</button> </div> </div>

Disclosure and feedback

HookBehavior
data-w-expand-toggleToggles the matching expansion panel.
data-w-speed-dialToggles a floating speed-dial action group.
data-w-alert-dismissDismisses the nearest alert.

Motion

HookBehavior
data-w-transition-toggleRuns a named transition on the matching target.
data-w-flipMarks a container whose child reorders should use FLIP animation.
data-w-flip-reorderReorders a FLIP container for no-JS-authoring demos and prototypes.
data-w-crossfadeAnimates a shared element between two keyed targets.
data-w-tweenTweens text, width, height, opacity, transform, CSS variables, or attributes.
data-w-springRuns a spring-like transform or value animation.

Programmatic helpers

The behavior script exposes WToast(message, duration) for transient feedback, WTheme for theme control, and WMotion for transitions, expansion, FLIP, crossfade, tween, and spring helpers.
