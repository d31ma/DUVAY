Autocompletes

w-autocomplete is a searchable dropdown. Use items for simple strings or JSON objects with item-title/item-value. Add multiple, chips, and closable-chips for multi-select, and clearable for a reset button.

Single and multiple

Hide selected

Add hide-selected so that items already chosen drop out of the dropdown — ideal for multi-select with chips. Removing a chip brings the option back.

Hide no data

Add hide-no-data to suppress the “no data” message when a search has no matches — useful when the list is populated asynchronously.
