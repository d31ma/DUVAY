Date inputs

w-date-input is a Vuetify-style date field: a text input with a calendar icon that opens a popup date picker. The canonical value is always ISO — use the format attribute to change how it reads.

Basic

Format and constraints

Set format to a pattern such as MM/dd/yyyy or dd.MM.yyyy. Use min and max to restrict selectable dates.

Clearable

Add the clearable boolean attribute to show a clear button.

Multiple and range

Set mode="multiple" to toggle several dates, or mode="range" to pick a start and end date.

API

  • value — ISO date value(s). Comma-separated for multiple; start,end for range.
  • label, hint, error, placeholder, size, disabled, readonly, name — standard field attributes.
  • min / max — ISO date bounds.
  • format — display pattern (yyyy, MM, M, dd, d).
  • mode — single (default), multiple, or range.
  • clearable — show a clear button.

Events: input while typing and change when the value commits.
