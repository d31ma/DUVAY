Infinite Scroller

w-infinite-scroll emits a load event as the user reaches an edge of a scrollable area. Fetch the next batch, then call done(status) with "ok" (more available), "empty" (stop), or "error" (show a retry). It supports automatic (intersect) and manual modes, loading from the start, end, or both edges, and a horizontal direction.

Wiring it up

Listen for load, append your data, then settle with done:

const scroller = document.querySelector('w-infinite-scroll');
let page = 0;

scroller.addEventListener('load', async (e) => {
  try {
    const rows = await fetchPage(++page);     // your fetch
    appendRows(scroller, rows);               // add to the slot content
    e.detail.done(rows.length ? 'ok' : 'empty');
  } catch {
    e.detail.done('error');                   // shows a Retry button
  }
});

Intersect (default)

In intersect mode the bottom sentinel auto-loads when it scrolls into view; a spinner shows while the fetch is in flight.

Sides & direction

Use side="start", "end" (default), or "both" to choose which edge loads, and direction="horizontal" for horizontal carousels. Set margin to start loading earlier (px before the edge).

Sentinel

<w-infinite-scroll-intersect> mirrors Vuetify’s internal sentinel as a standalone subcomponent for custom scroll containers — place it inside your own scroller and listen for its load event.
