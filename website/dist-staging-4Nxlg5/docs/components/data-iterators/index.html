Data Iterators

w-data-iterator renders records as repeated cards and owns the data pipeline — filter (search) → sort (sort-by) → paginate (items-per-page) — with a footer showing the item range, an optional per-page select, and pagination. It also surfaces loading and empty states. Pass records as a JSON array or compact title|subtitle|meta; … rows.
