Speed Dials

A floating action button that reveals related actions in a radial or linear menu. Supports hover, click, keyboard, and multiple transition and direction modes.

Basic

Directions

Use location to control the fan direction: top, bottom, left, right combined with start, end, or center.

Transitions

Choose scale (default), slide, or fade.

Open on Hover

Add open-on-hover to reveal actions on mouseenter and hide on mouseleave.

Hover over the FAB in the CSS column to see the effect (static example shown open).

Extended Actions

Child w-fab elements can use label for extended action buttons inside the speed dial.
