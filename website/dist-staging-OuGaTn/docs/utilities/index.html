Utilities

Single-purpose classes for spacing, layout, sizing, and more — compose them to build UI without writing CSS. Spacing uses a numeric scale where 1 = 4px, 2 = 8px, 4 = 16px, up to 16 = 64px.

Spacing

Margin (m) and padding (p) on any side: a (all), x, y, t, r, b, l. For example .w-pa-4, .w-mx-auto, .w-mt-2. Margins also accept auto and negatives (.w-ma-n2).

pa-2 pa-4 pa-6
ma-2 ma-4 ma-6

Flexbox

.w-flex plus alignment helpers — justify: .w-justify-start, .w-justify-center, .w-justify-between, .w-justify-around, .w-justify-evenly, .w-justify-end; align: .w-items-start, .w-items-center, .w-items-end, .w-items-stretch, .w-items-baseline.

Text truncation

Use .w-truncate for a single-line ellipsis, and .w-line-clamp-* for multi-line clamping.

This sentence is far too long to fit on one line, so it will be truncated with an ellipsis.

This paragraph spans multiple lines but is clamped to exactly two lines. Anything beyond that point is hidden and an ellipsis is shown at the end of the second line.
