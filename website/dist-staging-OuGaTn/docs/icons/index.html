Icons

w-icon renders an icon resolved through the DuVay icon registry, matching Vuetify’s VIcon API: size aliases, color, opacity, disabled, and start/end spacing helpers. The icon name is rendered as text by default; register an icon set (font, ligature, or SVG) to draw glyphs. See Icons setup.

Sizes

Use Vuetify sizes x-small … x-large, the xs…xl aliases, or any CSS length.

★ ★ ★ ★ ★ ★

Color

Pass a theme token (primary, success, warning, error…) or any CSS color.
