Floating Action Buttons

Prominent circular buttons for the primary action on a screen. DuVay FABs support Vuetify-style sizes, variants, colors, positioning, and extended labels.

Basic

Sizes

Use Vuetify-style sizes: x-small, small, default, large, and x-large.

Variants

Vuetify variant names: elevated (default), flat, tonal, outlined, text, and plain.

Colors

Positioning

Use fixed or absolute with location (or legacy position) to place the FAB in a corner.

Rounded

Active

The active attribute toggles the selected state styling.
