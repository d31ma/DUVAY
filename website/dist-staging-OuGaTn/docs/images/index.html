Images

w-img is a responsive image with Vuetify VImg parity: cover cropping, aspect-ratio reservation, gradient overlays, a blurred lazy-src placeholder, srcset/sizes, and overlaid slot content. It emits load and error events.

Cover and aspect ratio

cover crops the image to fill its box; pair with aspect-ratio to reserve space and prevent layout shift.

Gradient overlay

Overlay a CSS gradient — handy behind text on a hero image.

Lakeside
Lakeside

Lazy placeholder

lazy-src shows a tiny blurred image while the full src loads, then fades it out. Images load lazily by default; add eager to load immediately.
