Responsive Modes

responsive="auto" uses container queries to preserve the table on wide surfaces and stack labeled cells on narrow ones. Use stack to force records or scroll when horizontal comparison is essential. CSS-class tables use w-table-wrap--responsive-auto, a matching grid modifier, and data-label on each data cell.

Gridlines

Native <table> children accept gridlines="horizontal" (default), vertical, all, or none — matching Vuetify's v-table. CSS-class tables add w-table--gridlines-* to the <table>.
