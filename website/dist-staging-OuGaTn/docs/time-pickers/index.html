Time Pickers

w-time-picker is a standalone clock picker inspired by Vuetify's v-time-picker. It supports AM/PM and 24-hour display, seconds, allowed time windows, scroll-wheel editing, input fields, color, elevation, density, and compact headers while keeping the canonical value in 24-hour HH:MM or HH:MM:SS format.
