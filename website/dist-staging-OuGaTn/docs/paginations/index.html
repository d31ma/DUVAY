Pagination

Navigate large data sets with page controls, truncation, and accessible navigation.

Basic

Truncation with total-visible

When length exceeds total-visible, the middle is collapsed with an ellipsis.

… …

First and last buttons

Add show-first-last-page to render first («) and last (») navigation buttons.

Variants

Choose from text (default), flat, tonal, outlined, and elevated.

Sizes

Vuetify-style sizes: x-small, small, default, large, x-large.

Rounded

Use rounded="circle" (or rounded="pill") for fully rounded page buttons.

Color and active-color

Override the active page color with color or active-color.
