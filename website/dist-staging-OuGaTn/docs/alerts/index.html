Alerts

Contextual messages for feedback, warnings, and errors.

Types

Use type for contextual color and icon. Legacy variant="success" style syntax still works, but new alerts should prefer type.

i
Information
Something you should know about.
OK
Success
Your changes have been saved.
!
Warning
Please review before continuing.
x
Error
Something went wrong. Please try again.

Variants

Use variant for visual treatment: flat, tonal, outlined, text, plain, or elevated.

Flat
Default filled treatment.
Tonal
Subtler filled treatment.
Outlined
Transparent surface with accent border.
Elevated
Raised with the DuVay shadow token.
