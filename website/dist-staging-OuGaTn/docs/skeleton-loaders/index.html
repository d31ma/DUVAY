Skeleton loaders

w-skeleton-loader (alias w-skeleton) renders shimmering placeholders while content loads. Use type to compose bone presets — basic bones (text, heading, avatar, image, button, chip, divider…), @n repeats, and composites like list-item-avatar-two-line, article, actions, or card.
