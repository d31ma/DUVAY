Utilities

Single-purpose classes for spacing, layout, sizing, and more — compose them to build UI without writing CSS. Spacing uses a numeric scale where 1 = 4px, 2 = 8px, 4 = 16px, up to 16 = 64px.

Spacing

Margin (m) and padding (p) on any side: a (all), x, y, t, r, b, l. For example .pa-4, .mx-auto, .mt-2. Margins also accept auto and negatives (.ma-n2).

Text truncation

Use .text-truncate for a single-line ellipsis, and .w-line-clamp-* for multi-line clamping.
