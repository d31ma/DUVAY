Hover

w-hover tracks hover and focus state and exposes it for styling, with optional open-delay / close-delay. While hovered it adds is-hovering (and data-hovering="true") to its inner root and reflects an active attribute on the host, so plain CSS can react — no JavaScript needed in your markup.

Reacting to hover

Target the hovering state in CSS to raise elevation, shift color, or reveal actions.
