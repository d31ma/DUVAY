Theme Providers

w-theme-provider scopes a DuVay theme to a section of the page, matching Vuetify's VThemeProvider. It sets data-w-theme on its wrapper so descendant components resolve their tokens to that palette — useful for an always-dark sidebar or a previewed theme. By default the provider is renderless (it only scopes tokens); add with-background to paint the themed surface.

Scoping a theme

Set theme to light, dark, auto, or high-contrast. Components inside adopt that theme regardless of the page theme.
