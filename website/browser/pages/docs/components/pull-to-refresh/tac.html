Pull To Refresh

w-pull-to-refresh lets a scroll region be pulled down past a threshold to trigger a refresh, matching Vuetify's VPullToRefresh. When the threshold is crossed it fires load (and the back-compatible change) with a done callback; call done() or complete() when the async work finishes.

Pull gesture

Pull down from the top of the region (touch or click-drag) past pull-down-threshold to refresh.
