Defaults Providers

w-defaults-provider applies default attributes to descendant components, the attribute-driven analogue of Vuetify's VDefaultsProvider. Pass a defaults JSON object mapping a tag selector to the attributes it should receive. Defaults only fill in attributes a consumer hasn't set, and the nearest provider wins for nested scopes.

Nesting

Defaults merge across nesting: a nested provider overrides the keys it sets (here color) while keys it omits (variant) still inherit from the outer provider.
