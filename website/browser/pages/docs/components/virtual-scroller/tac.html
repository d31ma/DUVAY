Virtual Scroller

Virtual scrollers render only the items in (and near) the viewport, so a list of thousands stays smooth. Provide items and a height; set a uniform item-height for the fast path, or omit it for dynamically measured rows.

Basic

With a fixed item-height, only a small window of rows exists in the DOM at once. The CSS-class version shows the rendered window markup.

Dynamic item height

Omit item-height and each row is measured individually — useful when content wraps to different heights. scrollToIndex still lands precisely because offsets come from the measured sizes.
