Virtual Tables

w-data-table-virtual renders only the rows in (and near) the viewport, padding the rest with spacer rows so the scrollbar stays accurate. There’s no pagination — you scroll the whole, sorted dataset. Pair a fixed height with a uniform item-height. It shares data table’s headers, items, and sorting.

Basic

Thirty rows in a 240px window — only the visible slice (plus a small overscan) is in the DOM; the rest is one spacer row.

For large datasets, set items as a property in application code rather than inline:

const rows = Array.from({ length: 10000 }, (_, i) => ({ name: 'Row ' + i, value: i }));
document.querySelector('w-data-table-virtual').items = rows;
