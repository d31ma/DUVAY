Sparklines

w-sparkline draws a small inline trend line or bar chart from a list of values. It supports fills, smoothing, gradients, custom line width, labels, min/max clamping, and an auto-draw animation. Pass numbers via values (comma list or JSON array).

Trend & bar

Set type to trend (default line) or bar.

Fill & smooth

Add fill to shade the area under a trend, and smooth to round the corners.

Color

Set color to a token name (e.g. success) or any CSS color.

Gradient

Pass a comma list of colors to gradient (with optional gradient-direction).

Line width

Labels

Provide labels for custom captions, or show-labels to print the values themselves.

Auto-draw

Add auto-draw to animate the trend line on load; tune with auto-draw-duration (ms).
