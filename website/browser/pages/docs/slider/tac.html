Slider

A control for choosing a value along a track. Use the .w-slider class on a native <input type="range"> for the minimal version, or the <w-slider> web component for labels, ticks, value bubbles, theming, and vertical orientation. For two thumbs, see the range slider.

Basic

Label

Add a label; the current value shows below the track.

Sizes

Five sizes from xs to xl; omit for the default.

Steps

Set min, max, and step to constrain the values.

Thumb label

Add thumb-label to show the value on hover or focus, or thumb-label="always" to keep the bubble visible.

Ticks

Add ticks to mark every step, or tick-labels (pipe-separated) to label each one.

Color

Theme the fill and thumb with color, and the rail with track-color, using any token color name.

Disabled & readonly

disabled dims and freezes the slider; readonly freezes it without dimming.

Vertical

Set direction="vertical" to orient the slider vertically.
