Installation

DuVay has zero dependencies and no build step. Paste a hosted URL into your HTML and go.

Latest release

These URLs always serve the newest CalVer build from https://duvay.del.ma.

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.js" defer></script>

Pin to a version

Replace VERSION with any CalVer release you want to lock to (for example 26.24.12). Versioned URLs never change.

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.js" defer></script>

Web Components

To use the DuVay web component library, import the module after the stylesheet:

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.css"> <script type="module" src="https://duvay.del.ma/shared/assets/duvay/dist/duvay-wc.min.js"></script> <w-btn variant="filled">Click me</w-btn>

Selective CSS

For app bundles, import only the component CSS you use. Each component stylesheet imports the DuVay core layer.

@import "duvay-css/components/buttons.css"; @import "duvay-css/components/cards.css"; @import "duvay-css/components/dialogs.css";

If you prefer an explicit dependency chain, import core once and then component files.

@import "duvay-css/core.css"; @import "duvay-css/components/buttons.css"; @import "duvay-css/components/cards.css";
