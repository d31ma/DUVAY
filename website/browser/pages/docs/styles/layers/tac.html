CSS Layers

DuVay uses CSS cascade layers to make overrides predictable. Layers are ordered from least to most specific: reset, elements, components, utilities.

@layer w-reset, w-elements, w-components, w-utilities; @layer w-utilities { .text-center { text-align: center; } }

Author layers

Put your own styles in an author layer that comes after w-utilities, or use unlayered styles for intentional overrides.
