Borders

Utilities for adding or removing hairline (1px) borders on an element. DuVay also ships divider helpers for separating content horizontally or vertically.

Usage

All sides

Use w-border to add a 1px border around the whole element.

w-border
<div class="w-border pa-4">Bordered</div>

Individual sides

Apply a border to a single edge with w-border-bottom, w-border-left, or w-border-right.

bottom
left
right
<div class="w-border-bottom">Bottom border</div> <div class="w-border-left">Left border</div> <div class="w-border-right">Right border</div>

Remove borders

Use w-border-none to remove borders from an element that inherits them from a parent or component style.

Border removed
<div class="w-card w-border-none">No border</div>

Dividers

Use w-divider for a full-width horizontal divider, w-divider--vertical for a vertical one, and w-divider--inset for an inset horizontal divider.

Section above

Section below

<hr class="w-divider"> <hr class="w-divider--vertical"> <hr class="w-divider--inset">

Reference

ClassDescription
w-border1px solid border on all sides using --w-border.
w-border-bottom1px solid border on the bottom edge only.
w-border-right1px solid border on the right edge only.
w-border-left1px solid border on the left edge only.
w-border-noneRemoves all borders.
w-dividerFull-width 1px horizontal divider using --w-divider.
w-divider--vertical1px vertical divider that stretches to its container height.
w-divider--insetHorizontal divider inset from the leading edge.
