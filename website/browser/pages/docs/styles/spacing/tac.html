Spacing

Update your layout without creating new classes. Spacing helpers apply margin, padding, or gap to any element in 4px increments from 0 to 16, mirroring Vuetify's spacing helpers. Classes follow the format {property}{direction}-{size} — e.g. pa-4 (padding, all, 16px) or mt-2 (margin, top, 8px).

Usage

The property applies the type of spacing:

  • m — applies margin
  • p — applies padding
  • g — applies gap

The direction designates the side:

DirectionApplies to
t / b*-top / *-bottom
l / r*-left / *-right
s / einline-start / inline-end (RTL-aware — flips under RTL)
x / yleft+right / top+bottom
aall directions
r / c (gap)gr-* row-gap · gc-* column-gap · ga-* gap

The size is a 4px increment from 0 to 16 (0=0, 1=4px … 16=64px), auto (margins), or a negative margin n1–n16.

<div class="pa-4">Padding all sides, 16px</div> <div class="mx-auto">Centered with auto inline margins</div> <div class="ms-2 pe-6 ga-4">Logical start margin, end padding, gap</div>

Examples

Breakpoints

Apply margin or padding at a given breakpoint with {property}{direction}-{breakpoint}-{size} (breakpoints sm 600px, md 960px, lg 1280px, xl 1920px). The base size applies below the breakpoint; resize the window to see it change.

pa-2 pa-md-8
<div class="pa-2 pa-md-8">Tight on mobile, roomy from md up</div>

Negative margin

Margin can be applied negatively at the same 1–16 intervals with n1–n16 — useful for overlap.

Class reference

ClassDescription
ma-{0-16} / pa-{0-16}Margin / padding on all sides.
mx- / my- / px- / py-{0-16}Inline (x) / block (y) margin & padding.
mt- / mr- / mb- / ml-{0-16}Single-edge margin (and pt-/pr-/pb-/pl- for padding).
ms- / me- / ps- / pe-{0-16}Logical inline-start / inline-end margin & padding (RTL-aware).
ga- / gr- / gc-{0-16}Gap / row-gap / column-gap.
m{dir}-autoAuto margin (ma-auto, mx-auto, ms-auto, …).
m{dir}-n{1-16}Negative margins on any direction.
{property}{direction}-{sm|md|lg|xl}-{size}Responsive variants applied from that breakpoint up.
