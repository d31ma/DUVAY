Sizing

Sizing utility classes modify the dimensions of an element, mirroring Vuetify's sizing helpers. Apply height and width with the format .{prefix}-{size} — w for width, h for height. Each also has responsive variants (w-md-50, h-lg-100) that apply from that breakpoint up.

Usage

Set the width of a block-level element. Percentages are relative to the parent.

Responsive

Append a breakpoint — sm (≥600px), md (≥960px), lg (≥1280px), xl (≥1920px) — to apply a size from that width up: {prefix}-{breakpoint}-{size}. The box below is full width on mobile and half width from md.

w-100 w-md-50
<div class="w-100 w-md-50">Full width, then half from md up</div>
