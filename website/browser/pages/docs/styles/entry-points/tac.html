Entry points

DuVay provides full-bundle entry points for simple apps and split CSS entry points for apps that only want selected components.

FilePurpose
duvay.cssFlattened full CSS bundle with core and every component.
core.cssReset, tokens, themes, motion, typography, and utilities.
components/*.cssIndividual component styles, each importing core.css.
grid.css, forms.css, etc.Compatibility indexes that import related component CSS files.
duvay.jsBehaviour hooks for CSS-class usage (dropdowns, dialogs, etc.).
duvay-wc.jsCustom element definitions for the web-component API.

Hosted from duvay.del.ma

Use the latest release:

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.js" defer></script> <script type="module" src="https://duvay.del.ma/shared/assets/duvay/dist/duvay-wc.min.js"></script>

Selective CSS

Import individual component files when you want a smaller CSS surface. Component files import the required core layer themselves.

@import "duvay-css/components/buttons.css"; @import "duvay-css/components/cards.css"; @import "duvay-css/components/tooltips.css";

For explicit control, import core once and then the component files your app uses.

@import "duvay-css/core.css"; @import "duvay-css/components/buttons.css"; @import "duvay-css/components/cards.css";

Pin to a CalVer release

Every release is kept at a permanent URL. Replace VERSION with the CalVer you want (for example 26.24.12):

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.js" defer></script> <script type="module" src="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay-wc.min.js"></script>
