Colors

DuVay uses a semantic color system rather than a fixed Material palette. Each role — primary, secondary, tertiary, success, warning, error — resolves to a theme token that adapts across the light, dark, and high-contrast themes. Apply a role as a background with bg-* (which also sets a legible on-color) or as a foreground with text-*.

Tokens

Reference the underlying custom properties directly in your own CSS to stay theme-aware. Every role has a base color, a container variant, and matching on-colors for accessible contrast.

.my-banner { background: var(--w-primary-container); color: var(--w-on-primary-container); }

High contrast

The high-contrast theme increases color separation and border thickness for users who need stronger differentiation. Because these classes resolve to tokens, no markup changes are needed — switching the theme updates every color automatically.
