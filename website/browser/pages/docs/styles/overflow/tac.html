Overflow

Utilities for configuring how content that exceeds its container is handled. DuVay also provides a styled thin scrollbar helper.

Usage

Auto overflow

Use overflow-auto to add scrollbars only when content overflows the container.

Scrollable region. Add enough content to exceed the fixed height and an overflow scrollbar appears.

More content. More content. More content. More content. More content.

Even more content so the vertical scroll is visible in this example block.

<div class="overflow-auto" style="max-height: 5rem;"> ...scrollable content... </div>

Clip overflowing content

Use overflow-hidden to clip any content that exceeds the container bounds.

This content is clipped when it exceeds the fixed height of the container.

More content. More content. More content. More content. More content.

Even more content that should be clipped rather than scrolled.

<div class="overflow-hidden" style="max-height: 5rem;"> ...clipped content... </div>

Vertical-only scrolling

Use overflow-y-auto when you want vertical scrolling only. Pair with w-scrollbar for a thin themed scrollbar.

Vertical-only scroll region with a themed thin scrollbar.

More content. More content. More content. More content. More content.

Even more content so the scrollbar becomes visible.

<div class="overflow-y-auto w-scrollbar" style="max-height: 5rem;"> ...scrollable content... </div>

Reference

ClassDescription
