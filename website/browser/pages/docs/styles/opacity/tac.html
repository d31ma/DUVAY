Opacity

Utilities for controlling the opacity of an element. Use a fixed opacity step for static transparency, or combine opacity with a hover state to fade elements on interaction.

Usage

Opacity steps

Apply w-opacity-* to change an element's transparency. The DuVay scale uses 0, 50, 75, and 100.

100
75
50
0
<div class="w-opacity-50">Half transparent</div>

Hover opacity

Pair opacity with a custom hover rule to fade elements in or out when the user points at them.

.my-button { opacity: 1; transition: opacity var(--w-transition-fast); } .my-button:hover { opacity: 0.5; }
