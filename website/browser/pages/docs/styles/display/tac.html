Display

Display helpers control the CSS display value of an element, plus responsive visibility helpers that show or hide content at mobile, tablet, and desktop breakpoints.

Usage

Inline and block

Use d-inline, d-inline-block, and d-block to switch between inline and block flow.

inline inline inline-block
block
<span class="d-inline">inline</span> <span class="d-inline-block">inline-block</span> <div class="d-block">block</div>

Flex containers

Use d-flex and d-inline-flex to create flex formatting contexts. Combine with gap and alignment utilities to lay out children.

flex 1 flex 2
inline-flex 1 inline-flex 2
<div class="d-flex ga-3">...flex children...</div> <span class="d-inline-flex ga-2">...inline-flex children...</span>

Hide elements

Use d-none to remove an element from layout, or the responsive helpers to show/hide at specific breakpoints.

this is hidden (d-none)
hidden on mobile, visible on tablet+
mobile-only visible
<div class="d-none">Hidden everywhere</div> <div class="w-hide-mobile">Hidden on small viewports</div> <div class="w-show-mobile">Only on small viewports</div>
