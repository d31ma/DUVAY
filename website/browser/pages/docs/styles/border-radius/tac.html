Border radius

Utilities for controlling the corner rounding of an element. By default DuVay's radius tokens resolve to 0 (sharp corners); only rounded-pill and rounded-circle produce visible rounding. The named radius classes still cascade to nested components (cards, buttons, inputs, chips, and more) and will show rounding whenever the theme tokens are customized.

Usage

Named sizes

Use rounded-sm through rounded-xl to apply the theme radius tokens. These resolve to 0 in the default theme, so the boxes below stay square until you customize the radius tokens.

sm
md

<div class="rounded-sm">Small radius</div> <div class="rounded-lg">Large radius</div>

Pill and circle

Use rounded-pill for fully rounded ends and rounded-circle to make an element circular.

rounded-pill
circle
<div class="rounded-pill px-5 py-3">Pill</div> <div class="rounded-circle" style="width: 5rem; height: 5rem;">Circle</div>

No rounding

Use rounded-0 to remove rounding from an element that would otherwise inherit it from a component style.

rounded-0 overrides rounded-pill
<div class="w-card rounded-0">Sharp card</div>
