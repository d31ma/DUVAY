Position

Utilities for controlling the positioning of elements, mirroring Vuetify's position helpers. Apply the CSS position scheme with position-*, then pin edges with the inset helpers top-0, right-0, bottom-0, left-0 (or inset-0 for all four).

ClassProperties
position-staticposition: static
position-relativeposition: relative
position-absoluteposition: absolute
position-fixedposition: fixed
position-stickyposition: sticky
top-0 / right-0 / bottom-0 / left-0top/right/bottom/left: 0
inset-0inset: 0 — pin all four edges (DuVay)

Usage

Static

The default for all elements. The element follows normal document flow, and top/right/bottom/left have no effect.

position-static (offsets ignored)

Relative

position-relative positions the element relative to its normal spot, so edge offsets shift it from there (and it becomes a containing block for absolutely-positioned children).

in normal flow
position-relative, left offset
<div class="position-relative" style="left: 2rem">Shifted</div>

Absolute

position-absolute positions the element relative to its nearest positioned ancestor (here a position-relative container), removing it from normal flow.

relative container absolute · bottom-0 right-0 top-0 left-0
<div class="position-relative"> <span class="position-absolute bottom-0 right-0">Pinned</span> </div>

Sticky

position-sticky keeps an element in flow until the scroll position crosses a threshold (e.g. top-0), then it sticks. Scroll the box below.

position-sticky top-0 — scroll ↓

Fixed

position-fixed pins an element relative to the viewport, so it stays put while the page scrolls — useful for back-to-top buttons, snackbars, and floating action buttons. (Not shown live, since a fixed element escapes this preview to the viewport.)

<button class="position-fixed bottom-0 right-0 ma-4">Back to top</button>
