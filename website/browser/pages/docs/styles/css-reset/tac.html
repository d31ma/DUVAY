CSS Reset

DuVay’s reset is intentionally minimal. It sets box-sizing, removes default margins and padding from framework-owned elements, and establishes accessible baseline styles.

/* Scoped reset: only elements DuVay styles are reset */ .w-reset, [class^='w-'], [class*=' w-'] { margin: 0; padding: 0; }

Global rules

The reset also includes :focus-visible outlines and a prefers-reduced-motion guard. These apply everywhere for accessibility.

Bring your own base

If your project already has a base stylesheet, load it after duvay.css so it takes precedence.
