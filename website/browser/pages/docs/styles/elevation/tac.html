Elevation

Elevation helpers apply shadow depth to convey layering. Classes map to shadow tokens (--w-shadow-sm, --w-shadow-md, --w-shadow-lg) and cascade to nested components such as cards, buttons, sheets, and menus.

Usage

Elevation levels

Pick a level from elevation-0 (flat) to elevation-6 (highest). Intermediate levels share the same shadow tokens, giving you fine-grained semantic control.

0
1
3
6
<div class="w-card elevation-1">Resting card</div> <div class="w-card elevation-3">Elevated card</div>

Hover elevation

Increase elevation on hover by adding a custom rule. Combine with a transition token so the lift feels smooth.

Hover to lift
.my-card { box-shadow: var(--w-shadow-sm); transition: box-shadow var(--w-transition-normal); } .my-card:hover { box-shadow: var(--w-shadow-md); }

Remove shadows

Use w-shadow-none to explicitly remove a box shadow from a component that carries one by default.

Shadow
No shadow
<div class="w-card w-shadow-none">Flat card</div>
