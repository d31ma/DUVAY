Float

DuVay does not ship dedicated float utility classes. The framework favors flexbox and grid for layout, which avoid the clearing and collapse issues that come with floats. Use the flexbox alignment helpers instead.

Usage

Push one item to the end

Apply ml-auto to a flex child to push it to the end of the row — the equivalent of float: right without needing clearfixes.

Start
Pushed to the end
<div class="d-flex"> <div>Start</div> <div class="ml-auto">Pushed to the end</div> </div>

Align the whole row

Use justify-end or justify-space-between to align all children at once, and mr-auto to pin an item to the start.

End aligned 1
End aligned 2
Pinned start
End
End
<div class="d-flex justify-end">...all to the end...</div> <div class="d-flex"> <div class="mr-auto">Pinned start</div> <div>End</div> </div>

Alternatives

ClassDescription
ml-autoPushes a flex item to the end of the row (replaces float: right).
mr-autoPushes a flex item to the start (replaces float: left).
justify-endAligns all flex children to the end of the container.
justify-space-betweenDistributes children with the first at the start and last at the end.
