Cursor

Utilities for controlling the cursor shown when hovering an element. Pair cursor helpers with selection and pointer-event utilities for complete control over interaction affordances.

Usage

Pointer and default

Use cursor-pointer on clickable elements and cursor-default where you want to keep the normal arrow cursor.

cursor-pointer
cursor-default
<button class="cursor-pointer">Clickable</button> <div class="cursor-default">Normal cursor</div>

Text selection and pointer events

Add w-select-none to prevent text selection, or w-pointer-none to let clicks pass through to an element below.

selection disabled
pointer-events none overlay underlying content
<div class="w-select-none">Cannot select this text</div> <div class="w-pointer-none">Clicks pass through</div>

Reference

ClassDescription
cursor-pointerShows the pointer (hand) cursor on hover.
cursor-defaultShows the default arrow cursor.
w-select-nonePrevents text selection (user-select: none).
w-pointer-noneDisables pointer events on the element.
