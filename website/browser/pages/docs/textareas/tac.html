Textareas

w-textarea is the multi-line counterpart to the text field — same variants, density, labels, counter, clearable, and inner content, plus rows, auto-grow, max-rows, and no-resize. For a bare styled control use the .w-textarea class on a native <textarea>.

Variants

Set variant to outlined (default), filled, underlined, plain, or solo.

Auto-grow

Add auto-grow so the field expands with its content; cap it with max-rows.

No resize

Add no-resize to remove the drag handle (implied by auto-grow).
