Command palette

A search input over a list of <w-command-item>s. Type to filter; use ↑ ↓ to move, Enter to select, and Home / End to jump. Items can carry a leading icon, a subtitle, and a shortcut; selecting one emits change.

Grouped items

Add <div class="w-command-subheader"> labels and <div class="w-command-divider"> rules between items to organize the list. The same structure can be produced declaratively with the items array — an array of item objects (title, subtitle, icon, shortcut, value, disabled) plus subheader and divider rows.

Overlay & hotkey

Give <w-command> a hotkey (for example mod+k — ⌘ on macOS, Ctrl elsewhere) to open it as a centered overlay from anywhere. Press ⌘K / Ctrl K now, or use the button. Esc or a backdrop click closes it; open it programmatically with the open attribute or the show() / hide() methods.
