Color inputs

w-color-input is a compact field over the native colour input. Add swatches (a comma-separated hex list) for a clickable preset row, set mode to rgb or hsl to change how the value reads, and disabled to lock it. Picking a colour emits change. For a full panel with a live preview, see the color picker.
