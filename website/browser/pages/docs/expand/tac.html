Expansion Panels

Collapsible sections for progressive disclosure of content. Panels support grouped selection, mandatory state, and Vuetify-style visual variants.

Basic Panel

DuVay is a zero-dependency CSS framework for building polished interfaces with tokens, utilities, and Light-DOM web components.

Grouped Panels

Include duvay.css and optionally duvay-wc.js for the custom elements.

Override CSS variables or theme attributes to tune color, density, radius, and elevation.

Headers are buttons with visible focus states and expanded state communicated through ARIA.

Multiple Selection

Multiple panels can stay open together.

Omit mandatory when every panel should be allowed to close.

Variants

Active popout panels expand to the full row width.

Use inset when the open item should tuck inward.

Readonly and Disabled

Readonly panels communicate state without allowing toggles.

Disabled titles are unavailable to interaction.
