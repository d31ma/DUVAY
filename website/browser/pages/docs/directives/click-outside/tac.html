Click Outside

The DuVay equivalent of Vuetify's v-click-outside directive. Add w-click-outside to any element and DuVay fires a bubbling w-click-outside event whenever a click lands outside that element. Like Vuetify's directive, this is the same mechanism DuVay uses internally to dismiss dropdowns, menus, and dialogs. Requires duvay-directives.js to be loaded.

Usage

Mark the element you want to watch with w-click-outside, then listen for the w-click-outside event to run your handler. The element itself and all of its descendants are always treated as “inside” — clicking them will not fire the event.

<div id="panel" class="w-card" w-click-outside>
  <p>Click anywhere outside this card to close it.</p>
</div>

<script>
  const panel = document.getElementById('panel');
  panel.addEventListener('w-click-outside', () => {
    panel.classList.remove('is-open');
  });
</script>

Options

The directive's behavior is configured through companion w-* attributes. The table below maps each one to the Vuetify modifier it replaces.

DuVay attributeVuetify equivalentDescription
w-click-outside v-click-outside="handler" Marks the target element. Fires w-click-outside when a click occurs outside of it.
w-click-outside-include="#a, #b" :include option A comma-separated CSS selector list of extra elements treated as “inside.” Clicks on these elements (or their descendants) will not fire the event — useful for an external trigger button.

Vuetify also exposes a closeConditional option, a function that decides whether the handler should run. DuVay has no attribute form for this; gate the behavior inside your own w-click-outside listener instead.

Events

The directive dispatches a single bubbling CustomEvent named w-click-outside on the target element. Its detail.originalEvent holds the underlying DOM click event.

el.addEventListener('w-click-outside', (event) => {
  const original = event.detail.originalEvent;
  console.log('Clicked outside at', original.clientX, original.clientY);
});

Example

A dismissible panel with an external toggle button. The button is listed in w-click-outside-include so clicking it does not immediately re-close the panel that it just opened.

<button id="toggle" class="w-btn w-btn-outlined">Toggle panel</button>

<div
  id="panel"
  class="w-card"
  w-click-outside
  w-click-outside-include="#toggle"
>
  <p>Click outside (but not on the toggle) to close me.</p>
</div>

<script>
  const toggle = document.getElementById('toggle');
  const panel = document.getElementById('panel');

  toggle.addEventListener('click', () => {
    panel.classList.toggle('is-open');
  });

  panel.addEventListener('w-click-outside', () => {
    panel.classList.remove('is-open');
  });
</script>
