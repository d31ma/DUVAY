Resize

The DuVay equivalent of Vuetify's v-resize directive. Add w-resize to any element to call your code whenever that element changes size. Vuetify's v-resize listens to the window resize event only; DuVay is a superset — it observes the element itself with a ResizeObserver and listens to window resize, then dispatches a bubbling w-resize event enriched with the element's current dimensions. Requires duvay-directives.js.

Usage

Mark the element with w-resize and listen for the w-resize event. It fires once immediately on wiring (so your handler runs with the initial size) and again on every subsequent size change. Resize your window or the element and observe the values change.

<div class="responsive-box" w-resize>
  Width: <span class="value"></span>
</div>
const box = document.querySelector('.responsive-box');
box.addEventListener('w-resize', (e) => {
  box.querySelector('.value').textContent = Math.round(e.detail.width) + 'px';
});

Modifiers

By default w-resize calls your handler immediately when it is wired up, matching Vuetify's default behavior. Add the w-resize-quiet option to suppress that initial call — the equivalent of Vuetify's .quiet modifier — so your handler only runs on actual size changes.

OptionDescription
w-resize-quietSuppress the immediate call that fires when the directive is wired up. The handler then runs only on subsequent resize events.
 
<div class="responsive-box" w-resize w-resize-quiet>
  <!-- w-resize fires only on actual size changes, not on wiring -->
</div>

Events

The directive dispatches a bubbling w-resize CustomEvent. Its detail carries the element's current bounding box plus the raw observer entries when available.

PropertyTypeDescription
detail.widthnumberThe element's current bounding-box width in pixels.
detail.heightnumberThe element's current bounding-box height in pixels.
detail.entriesResizeObserverEntry[]The raw observer entries, when the resize came from the ResizeObserver. Absent for window-driven calls.
 
el.addEventListener('w-resize', (e) => {
  const { width, height, entries } = e.detail;
  console.log('size is now', width, 'x', height);

  if (entries) {
    for (const entry of entries) {
      // entry.contentRect, entry.borderBoxSize, etc.
    }
  }
});

Example

Toggle a class based on the element's own width — container-driven responsiveness without media queries.

<div class="card" w-resize>
  <h3>Adaptive card</h3>
  <p>Resize the container to switch layouts.</p>
</div>
const card = document.querySelector('.card');
card.addEventListener('w-resize', (e) => {
  card.classList.toggle('is-narrow', e.detail.width < 480);
});
