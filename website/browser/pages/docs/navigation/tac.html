DuVay provides primitives for topbars, sidebars, breadcrumbs, tabs, drawers, and mobile bottom navigation. All are built from utility classes — no JS required for the layout itself.

Sidebar (CSS only)

Use the .w-shell, .w-topbar, .w-sidebar, and .w-main classes to compose a full app frame. See the install page for a quick scaffold.

Mobile bottom nav

Add .w-mobile-nav at the page root. It only renders on screens narrower than 760 px and respects the device safe-area inset.

<nav class="w-mobile-nav">
  <a class="w-sidebar-folder active" href="/inbox">Inbox</a>
  <a class="w-sidebar-folder" href="/search">Search</a>
  <a class="w-sidebar-folder" href="/me">Profile</a>
</nav>
