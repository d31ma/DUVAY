Button toggles

w-btn-toggle groups a set of w-btn children into a single segmented control and manages which are selected. Clicking a button reports its value through a change event. It builds on the joined .w-btn-group visual.

Single selection

By default one button is selected at a time. Bind the chosen value with value. Add mandatory to always keep one selected.

Multiple selection

Add multiple to allow several selections; the value becomes a comma-separated list. Use max to cap how many can be active at once.

Divided

Add divided to draw a separator between buttons.

Variant & color

Set variant, color, or density on the toggle and they cascade to any child button that doesn't set its own.

Button group

For a static, non-selecting cluster, use .w-btn-group directly. Mark the chosen one with active, add .w-btn-group--vertical to stack, or .w-btn-group--divided for separators.
