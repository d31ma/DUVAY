Text fields

w-text-field is the full single-line control, mirroring Vuetify’s v-text-field: a floating label, five variants, density, prefix/suffix, inner slots, clearable, a character counter, a loading bar, and the native HTML5 validation attributes. The web component writes the .w-text-field-* markup shown in the CSS panes. For a bare styled input, see inputs.

Basic

Variants

Set variant to outlined (default), filled, underlined, plain, or solo.

Input types

Set type to any native value — email, password, url, tel, and more.

Density & sizes

density tightens vertical rhythm; size scales the whole control.

Prefix, suffix & inner content

Add static prefix / suffix text, name icons with prepend-inner-icon / append-inner-icon (resolved through the icon registry), or drop arbitrary content into the prepend-inner / append-inner slots.
