About

DuVay is a compact design framework for teams that want a Vuetify-like surface without taking on a Vue dependency or a runtime-heavy component stack.

Principles

  • Stay HTML-first: static markup should look correct before JavaScript enhances it.
  • Keep styling token-driven: components consume semantic variables, not one-off colors or spacing.
  • Expose Light DOM: web components should remain inspectable, themeable, and easy to override.
  • Prefer native browser behavior: add JavaScript only where it improves interaction or accessibility.

Relationship to Vuetify

Vuetify is the north star for coverage, naming familiarity, and Material Design concepts. DuVay adapts those ideas into a framework-agnostic CSS and web-component library with its own visual language.
