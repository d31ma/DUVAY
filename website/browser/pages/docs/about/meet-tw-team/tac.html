Meet the team

DuVay is maintained by a small group of design-systems engineers and contributors who care about lightweight, accessible UI.

Maintainers

The core team reviews all changes to CSS tokens, web components, and the documentation site. Reach out via GitHub issues and discussions.

Contributors

Every issue report, docs improvement, and pull request helps. See the repository’s contributors graph for the full list.
