Code of conduct

The DuVay project is committed to providing a welcoming, inclusive experience for everyone. This code applies to all project spaces: issues, pull requests, discussions, and chat.

Our standards

Be respectful and constructive. Welcome newcomers, accept constructive criticism, and focus on what is best for the community. Harassment, trolling, and discriminatory language are not acceptable.

Reporting issues

If you experience or witness unacceptable behaviour, email the maintainers or open a private issue. All reports will be handled confidentially.

Scope

This code governs participation in the DuVay repository and any public forum where the project is represented.
