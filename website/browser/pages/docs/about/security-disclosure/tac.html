Security disclosure

If you discover a security issue in DuVay, please report it privately so we can fix it before public disclosure.

How to report

Do not open a public issue for security vulnerabilities. Instead, email the maintainers directly or use GitHub’s private vulnerability reporting feature on the repository.

What to include

  • A clear description of the issue and the affected files or components.
  • Steps to reproduce, or a minimal test case if possible.
  • Your assessment of impact and any suggested remediation.

Response process

We will acknowledge receipt within a few business days, investigate, and release a patch as soon as a fix is ready.
