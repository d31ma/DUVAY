TailwindCSS

DuVay and Tailwind can coexist. Use DuVay for semantic components and design tokens, and Tailwind for highly custom, one-off layouts.

/* tailwind.config.js */ module.exports = { theme: { extend: { colors: { primary: 'var(--w-primary)', }, }, }, }

Token sharing

Map Tailwind’s theme keys to DuVay CSS custom properties so spacing, color, and radius stay consistent across both systems.
