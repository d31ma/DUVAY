UnoCSS + Vuetify preset

If you are migrating from Vuetify, an UnoCSS Vuetify preset can reproduce familiar utility names while DuVay provides the component layer.

// uno.config.ts import { presetVuetify } from 'some-migration-preset'; export default { presets: [presetVuetify()], };

Migration path

Use the preset during transition, then gradually replace Vuetify utility names with DuVay utilities or design tokens.
