UnoCSS + presetWind4

UnoCSS can generate Tailwind-compatible utilities from DuVay’s token values. Point UnoCSS at the same CSS custom properties to keep classes in sync.

// uno.config.ts import presetWind4 from '@unocss/preset-wind4'; export default { presets: [presetWind4()], theme: { colors: { primary: 'var(--w-primary)', }, }, };

Use with DuVay

Import duvay.css first, then let UnoCSS generate atomic utilities on top. DuVay components continue to use their own semantic classes.
