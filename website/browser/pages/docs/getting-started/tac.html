Minimal HTML

Use the hosted URLs to get started without a build step:

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.js" defer></script> <button class="w-btn w-btn-filled">Continue</button>

Pin to a version

Replace VERSION with the CalVer release you want. Versioned URLs are permanent.

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.js" defer></script> <button class="w-btn w-btn-filled">Continue</button>
