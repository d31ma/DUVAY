Introduction

DuVay is a zero-dependency design system for building clean, accessible interfaces with plain CSS, optional behavior hooks, and Light DOM web components.

What DuVay provides

Design tokens

Theme, spacing, shape, elevation, typography, and motion values are exposed as CSS custom properties.

Two authoring modes

Use DuVay as CSS classes in static HTML, or import duvay-wc.js for Vuetify components.

Progressive behavior

Dialogs, drawers, dropdowns, expansion panels, alerts, themes, and search use small data-w-* hooks.

Authoring model

Every component starts as semantic HTML and DuVay classes. Web components generate the same public structure, so teams can mix both approaches without changing the visual system.
