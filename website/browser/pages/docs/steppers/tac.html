Steppers

w-stepper guides a user through an ordered sequence of steps. It tracks an active step (value), marks earlier steps complete, optionally drives a w-stepper-window's panels, and wires w-stepper-actions (Back / Next).

Basic

Headers go in a w-stepper-header, panels in a w-stepper-window, and navigation in w-stepper-actions. The Next / Back buttons move the active step.
