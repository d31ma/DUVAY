Routing

DuVay does not ship a router. It renders standard links and buttons, so any client-side router — Vue Router, React Router, SvelteKit, or a small in-house solution — can drive navigation.

Native links

DuVay linkable components accept an href attribute and render a real <a> tag. This gives you right-click, middle-click, keyboard activation, and screen-reader support for free.

Client-side routers

Most routers intercept clicks on internal links. Because DuVay renders native anchors, the router's global link handler works without adapters. If your router requires a special component (e.g. <RouterLink>), wrap DuVay components inside it.

Vue Router example

<!-- The router intercepts the native click and does a SPA navigation. --> <RouterLink to="/docs/items" custom v-slot="{ href, navigate }"> <w-btn variant="filled" :href="href" @click="navigate"> Go to Items </w-btn> </RouterLink>

React Router example

import { useNavigate } from 'react-router-dom' function HomeLink() { const navigate = useNavigate() return ( <w-btn variant="filled" onClick={() => navigate('/docs/items')}> Go to Items </w-btn> ) }

Vanilla example

<w-btn id="next" variant="filled" href="/docs/items"> Go to Items </w-btn> <script> document.getElementById('next').addEventListener('click', (e) => { // Your tiny router decides whether to handle the URL locally. if (myRouter.matches(e.target.href)) { e.preventDefault() myRouter.navigate(e.target.href) } }) </script>

Navigation components

App bars, bottom navigation, breadcrumbs, tabs, and lists all accept href (or an internal link child) so they participate in normal browser navigation. Use your router's link component as the child when you need framework-specific routing behavior.

Active state

DuVay leaves active-state logic to your router. Pass active, value, or an explicit active class based on the current route. The docs site uses the same pattern: the sidebar compares the current path to each link's href and adds the active class.

<w-bottom-navigation :value="currentRoute"> <w-bottom-nav-item value="/" href="/">Home</w-bottom-nav-item> <w-bottom-nav-item value="/docs" href="/docs">Docs</w-bottom-nav-item> </w-bottom-navigation>
