Aspect Ratios

w-aspect-ratio reserves a fixed width-to-height ratio for any media or content, so layouts stay stable before images load. It mirrors Vuetify’s VResponsive / VAspectRatio: set ratio (or aspect-ratio), plus optional dimension and content-class props.

Default (16 / 9)

Common ratios

Pass any ratio as "w / h" or a number. Dimension props (width, height, max-width, …) accept a number (px) or any CSS length.

Embeds

Any media works — video, iframes, canvas. Content stretches to fill the ratio box.
