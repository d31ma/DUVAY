Animations

DuVay motion combines named transition helpers with Svelte-inspired FLIP, crossfade, tween, and spring helpers. Every helper respects prefers-reduced-motion.

API Reference

HelperPurpose
WMotion.enter(el, name)Shows an element with a named transition.
WMotion.leave(el, name)Hides an element with a named transition.
WMotion.toggle(el, open, name)Switches between enter and leave.
WMotion.setExpand(panel, open)Measures and animates expansion-panel height.
WMotion.flip(container, mutate)Runs FLIP animation around a DOM mutation.
WMotion.crossfade(from, to)Animates a cloned shared element between two targets.
WMotion.tween(el, options)Interpolates text, dimensions, transforms, attributes, or CSS variables.
WMotion.spring(el, options)Applies a damped spring-style value animation.
