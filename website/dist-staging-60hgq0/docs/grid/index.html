Grid

A 12-column flexbox grid for responsive layouts. Use w-container, w-row, and w-col for component layouts, or the equivalent .w-container, .w-grid-row, and .w-grid-col-* classes.

Container

Centers content and applies a max width. Add --fluid for full width, or --sm / --md / --lg / --xl for a fixed maximum.

<div class="w-container"> <!-- centered, max 1200px --> </div> <w-container> <!-- centered, max 1200px --> </w-container>
