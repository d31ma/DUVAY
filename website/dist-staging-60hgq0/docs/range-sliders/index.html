Range sliders

w-range-slider selects a range between two values with a pair of thumbs on a single track; the selected segment is highlighted. Bind the ends with start and end, and constrain with min, max, and step. For a single value, use the slider.

Basic

Min, max, and step

Constrain the range with min / max and snap to increments with step.

Disabled

Thumb labels

Add thumb-label to show each thumb’s value on hover or focus, or thumb-label="always" to keep the bubbles visible.

Ticks

Add ticks to mark each step along the track.

Vertical

Set direction="vertical" to orient the slider vertically.

Attributes

AttributeTypeDefaultDescription
min, maxnumber0 / 100Range bounds.
start, endnumbermin / maxCurrent range ends.
stepnumber1Increment between values.
labelstring—Field label.
directionstringhorizontalhorizontal or vertical.
thumb-labelboolean | alwaysfalseShow a value bubble over each thumb (always keeps it visible).
ticksbooleanfalseDraw a tick at every step.
disabledbooleanfalseDisable both thumbs.

Events

EventDetailDescription
w-input{ start, end }Fired while dragging either thumb.
w-change{ start, end }Fired when a thumb is released.
