OTP Input

w-otp-input (alias w-otp) renders one box per character for entering verification codes, mirroring Vuetify’s v-otp-input. Set the box count with length. Pasting a full code into any box distributes it across the row, and arrow keys move between boxes. It emits w-input as the value changes and w-complete once every box is filled.

Basic

Type

Set type to number to accept digits only, or password to mask each character.

Divider

The divider attribute draws a character between each box.

Disabled

Attributes

AttributeTypeDefaultDescription
lengthnumber6Number of character boxes.
value / model-valuestring—Current code.
typestringtexttext, number (digits only), or password (masked).
dividerstring—Character drawn between boxes.
placeholderstring—Placeholder shown in empty boxes.
disabledbooleanfalseDisable every box.

Events

EventDetailDescription
