Number inputs

w-number-input is a numeric field with stepper controls, mirroring Vuetify’s v-number-input. Use min, max, and step to constrain the value, precision and grouping to format it, and control-variant to change the stepper layout. It emits w-change (and update:model-value) as the number updates.

Basic

Control variants

The control-variant attribute sets the stepper layout: default (buttons grouped at the end), stacked (vertical spinner), split (− and + on opposite ends), or hidden (no buttons).

Formatting

Set precision for a fixed number of decimal places, grouping for thousands separators, and decimal-separator / group-separator to localize them. The value attribute always stays canonical (plain number); formatting applies only to what is shown.

min-fraction-digitsnumber—Minimum decimals to keep.
groupingboolean | stringfalseThousands separators: true, always, auto, min2.
decimal-separator, group-separatorstring. / ,Single-character separator overrides.

disabled, readonlybooleanfalseDisable the field and its controls.

Events

EventDetailDescription
w-input{ value }Fired on each keystroke with the parsed number.
w-change{ value }Fired on commit or step with the clamped number (or null).
update:model-value{ value }Mirror of w-change.
