Radio buttons

w-radio-group wraps a set of w-radio options for a single choice. The group manages the name, selected value, and keyboard navigation. Add inline to lay the options out in a row.

Basic

Delivery speed

Inline

The inline attribute arranges the options horizontally, wrapping as needed.
