Breadcrumbs

Breadcrumbs show the user's location within a hierarchy and provide a path back to parent pages. DuVay breadcrumbs support a declarative items array, custom dividers, leading and per-item icons, color theming, density, and rounded containers.

The separator between crumbs is drawn by CSS, so plain markup, the items array, and composed children all get automatic dividers. An explicit <w-breadcrumbs-divider> between two crumbs overrides the automatic one.

Basic

Mark the current page with the active class (or the active attribute), which renders a non-link crumb with aria-current="page".

Declarative items

For data-driven breadcrumbs, set an items array on <w-breadcrumbs> from JavaScript instead of composing children. Each entry is an object with a title (or text), an optional href, and optional disabled and icon values; the last entry becomes the current page automatically. Color and active color

color sets the link color and active-color the current-page color, using DuVay palette tokens. With CSS classes, set the --w-breadcrumb-color and --w-breadcrumb-active-color custom properties.

Background and rounded

Give the strip a tinted, rounded container with bg-color (a palette token) and rounded. With CSS classes, add w-breadcrumbs--rounded and set --w-breadcrumb-bg.

Use density (comfortable or compact) to adjust the vertical padding. With CSS classes, add w-breadcrumbs--comfortable or w-breadcrumbs--compact.

Custom divider element

For a one-off divider — an icon, a different glyph per position — place <w-breadcrumbs-divider> between crumbs. It overrides the automatic separator for that gap; provide the divider content as its child.

Accessibility

Use aria-label="Breadcrumb" on the wrapper and aria-current="page" on the current item. The <w-breadcrumbs> component adds both automatically, marks disabled crumbs with aria-disabled, and renders icons as aria-hidden.

API reference

Components: <w-breadcrumbs>, <w-breadcrumb>, <w-breadcrumbs-item> (alias of <w-breadcrumb>), <w-breadcrumbs-divider>.

<w-breadcrumbs> attributes: items, divider, icon, color, active-color, bg-color, density, rounded, disabled.

<w-breadcrumb> attributes: href, active, disabled, icon.
