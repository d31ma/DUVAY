Data Tables: Data and Display

Shaping the data and how it’s shown in w-data-table: rich column definitions, row selection, expandable detail rows, density, and loading / empty states. See Basics for sorting, search, and pagination.

Rich headers

For control over each column, set the headers (and items) property to JSON objects with title, key, align, sortable, and width. Objects are matched to columns by key. (Set these in JS rather than as attributes, since JSON in markup is awkward to escape.)

const table = document.querySelector('w-data-table');
table.headers = [
  { title: 'Dessert', key: 'name' },
  { title: 'Calories', key: 'calories', align: 'end' },
  { title: 'Type', key: 'type', sortable: false },
];
table.items = [
  { name: 'Frozen yogurt', calories: 159, type: 'Frozen' },
  { name: 'Eclair', calories: 262, type: 'Pastry' },
];

Selectable rows

Add show-select for a checkbox column with a select-all header. Use item-value to identify rows and select-strategy="single" for single selection. Listen for update:selected.

Add show-expand for a toggle that reveals a detail row listing every field. Listen for update:expanded.
