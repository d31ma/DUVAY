Combobox

w-combobox extends w-autocomplete to accept free-text values that are not in the list. Press Enter, or leave the field, to commit whatever was typed.

Basic

Delimiters and chips

Set delimiters so that typing one of those characters commits the current text as a value — each character in the attribute is a delimiter, so delimiters=", " ends a tag on comma or space. Combine with multiple, chips, and closable-chips for a tag editor.
