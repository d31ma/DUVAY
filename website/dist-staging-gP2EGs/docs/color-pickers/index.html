Color Pickers

w-color-picker provides a Vuetify-style HSV canvas, hue slider, optional alpha slider, editable hex value, preview, and preset swatches. For a compact field, use color inputs instead.

Alpha and Swatches

Use show-alpha for transparent colors. Pass swatches as a comma-separated or JSON list of hex values.
