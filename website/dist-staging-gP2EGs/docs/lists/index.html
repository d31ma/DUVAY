Lists

Vertical collections for navigation, settings rows, messages, and selectable options.

Basic List

w-list-item-title, w-list-item-subtitle, w-list-subheader, w-list-children, w-list-img, w-list-item-action, and w-list-item-media are available as Vuetify-named subcomponents.

Items API

items accepts a JSON array with item, divider, subheader, and nested children entries. Selection attributes also accept JSON arrays for multi-value state.

Tree view

w-treeview renders hierarchical, collapsible data. Pass a flat items array of >-delimited paths and DuVay builds the tree, or compose w-treeview-item children directly. Add open-all to expand every branch; toggle buttons are keyboard-focusable and report aria-expanded.
