Download

DuVay ships in two flavours that share one design system: CSS classes (style with utility and component classes) and optional web components (drop-in custom elements). Both are zero-dependency. Use the hosted URLs from https://duvay.del.ma, download the files, or pin to a specific CalVer release.

Quick start (hosted)

Paste into your <head>. The minified builds are served from this site and updated with every release.

CSS classes

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.js" defer></script>

Selective CSS

@import "duvay-css/core.css"; @import "duvay-css/components/buttons.css"; @import "duvay-css/components/tooltips.css";

Web components

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/dist/duvay.min.css"> <script type="module" src="https://duvay.del.ma/shared/assets/duvay/dist/duvay-wc.min.js"></script> <w-btn variant="filled">Click me</w-btn>

Pin to a CalVer release

Every release is kept at a permanent URL. Swap VERSION for the CalVer you want (for example 26.24.12).

<link rel="stylesheet" href="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.css"> <script src="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay.min.js" defer></script> <!-- optional web components --> <script type="module" src="https://duvay.del.ma/shared/assets/duvay/versions/VERSION/dist/duvay-wc.min.js"></script>

Versioned URLs never change, so you can upgrade on your own schedule by updating the version segment.

Direct download

Self-hosted builds — full (readable) and minified (production). Right-click → Save, or click to download.

AssetFullMinified
CSS framework duvay.css · ~307 KB duvay.min.css · ~231 KB
CSS core core.css · ~70 KB core.min.css · ~50 KB
Component CSS components/buttons.css components/tooltips.css
Behaviour layer (duvay.js) duvay.js · ~13 KB duvay.min.js · ~6.8 KB
Motion add-on duvay-motion.js · ~18 KB duvay-motion.min.js · ~10 KB
Web components (bundled) duvay-wc.js · ~417 KB duvay-wc.min.js · ~291 KB

The behaviour layer (duvay.js) wires CSS-class interactions — dialogs, dropdowns, toasts, theme cycling — through data-w-* attributes. The web-component bundle is self-contained and does not require it.

Versions

Releases follow CalVer (YY.WW.DD). Replace VERSION with the CalVer release you want; every release is permanently available at /shared/assets/duvay/versions/VERSION/dist/. Always test an upgrade against your app before moving the version pin.
