Switches

w-switch is a boolean toggle — an accessible checkbox styled as an on/off control. Use it for settings that take effect immediately. Style a native checkbox with the .w-switch classes, or use the <w-switch> web component for colors, sizes, loading, hints, and the inset/flat variants.

Basic

Colors

Set color to tint the track and accent when on.

Sizes

Four sizes from xs to lg; omit for the default (md).

Inset

Add inset so the track fully wraps the thumb.

Loading

Add loading to show a spinner in the thumb and block toggling while a request is in flight.

Hint & error

Add hint for helper text, or error to surface a validation message and tint the control.
